# 鹏翔驾校常见问题解答（FAQ）

## 1. 报名费用是多少？

**小车C1暑期优惠价**：3490 元
**小车C2暑期优惠价**：3390 元

驾校收取的这部分费用包含了：
- 报名费
- 受理费
- 保险费
- 计时培训费
- 科目二、科目三所有的补考费
- 考前模拟费

**另外需要缴纳的费用**：
- **520 元考试费**：科目三考完以后交给车管所
- **44 元体检费**：报名时在自助体检机上缴纳

**总费用**：
- C1：3490 + 520 + 44 = **4054 元**
- C2：3390 + 520 + 44 = **3954 元**

## 2. 驾校在什么地方？

我们鹏翔驾校目前有 **四个校区**、**两个考场**：

**校区**：
1. **南校区**：长安区新韦斗路
2. **汉城湖公园校区**：汉城湖公园北门
3. **秦汉新城考务中心**
4. **沣西训练场地**

**两个考场**：
- 南校区考场（长安区）
- 秦汉考务中心考场

## 3. 你们这个费用是包过吗？

科目二、科目三的补考费，驾校全部承担，也就是包你考过。

我们签订正规培训合同，一费制到底。科二科三补考费全部包含，包括考前的模拟训练。就算过来复训，一样是免费的，不收任何额外费用。

平时练车我们有 **免费大巴车接送**，一天三趟，20条班车线路图，覆盖整个西安市、咸阳、秦汉，包括沣西。

班车服务特点：
- 免费大巴车，一天三趟
- 20条班车线路，覆盖全西安市、咸阳、秦汉、沣西
- 点对点上门接送，不用自行奔波通勤
- 早晚班次、夜训班次全覆盖

具体班车路线可以拨打班车服务电话咨询：15934853221

还有就是我们是自己的考场，考试好预约，通过率高。练车单人单车，随到随学，不存在预约不上、排队练车的情况。

只用拿着身份证过来就行，在我们这儿做个体检、录系统、面签，一站式搞定，非常方便。
